crichez.secureboot.uki_config role – UKI Configuration

Note

This role is part of the crichez.secureboot collection (version 1.0.0).

It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it use: ansible-galaxy collection install crichez.secureboot.

To use it in a playbook, specify: crichez.secureboot.uki_config.

New in crichez.secureboot 1.0.0

Synopsis

  • Configure kernel-install to build and sign UKIs

Parameters

Parameter

Comments

uki_config_cmdline

path

The path to the kernel command line.

Default: "/etc/kernel/cmdline"

uki_config_initrd_generator

string

The tool kernel-install calls to generate an initramfs.

Default: "dracut"

uki_config_kernel_install_config_root

path

The path where configuration files are stored.

This should usually be `/etc/kernel` (the default), but you may wish to use `/usr/lib/kernel` instead.

Default: "/etc/kernel"
